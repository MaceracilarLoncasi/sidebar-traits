# D&D 5e Action Sheet

**D&D 5e Action Sheet** is a Foundry VTT module that organizes all **Actions, Bonus Actions, and Reactions** into a single, clean, and readable panel on the character sheet.

It is designed to answer one simple question during play—especially in combat:

> *“What can I do right now?”*

---

## ✨ Features

- Clear separation of **Actions / Bonus Actions / Reactions**
- Automatically lists:
  - Weapons
  - Spells
  - Items
  - Features
